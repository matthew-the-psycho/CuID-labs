1. Реализовать одностороннюю функцию быстрого возведения в степень.
2. Реализовать трассировку при выполнении программы.
3. С помощью переменной счётчика определить фактическое количество выполненных умножений.
4. НЕ использовать С++ :: #include <cmath> :: pow(base, power);

Реализовано:
1. Реализована функция одностроннего возведения в степень; //ENCRYPTOR
2. Реализована трассировка при выполнении программы (необходимая информация выводится в консоль);
3. Фактическое количество умножений подсчитывается по меткам трассировки (если флаг used == 1, 
    то учитывается два умножения, иначе одно :: для подробного понимания, смотреть в код функции ENCRYPTOR);
4. Для упрощения рассмотрения результатов результаты записаны в .log-файлы, 
    в соответствии с режимами работы программы.

main.cpp :: 
<.exe/.elf> <file to process> <--code or --decode> <--type-rand/--type-fix> <--regenerate-key>

Примечание :: 
В случае флага --decode флаг типа не влияет, используется тот ключ, который есть.
\\Совместимость типов/Совместимость вызова программы.
\\Так как идёт моделирование обмена через полученный симметричный секретный ключ, 
    полученный по системе Диффи-Хелмана.
