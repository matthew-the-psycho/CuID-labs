1. Написать программу, которая по заданному файлу 
    определяет частоты появления символов в нём.
2. Усовершенствовать программу 1 :: 
    чтобы она вычисляла энтропию файла по вычисленным частотам.
3. Сгенерировать несколько различных файлов ::
    ;: Символьно-наполненный :: 0 и 1;
    ;: Символьно-наполненный :: 0, 1 и ' ';
    ;: Символьно-наполненный :: 0x00 до 0xFF;
    :: Убедиться, что в случае большого количества одинаковых символов энтропия близится к 0
    :: А в случае случайных символов энтропия близится к log_2(Размер алфавита);

Реализовано:
Пункты 1 и 2 были объединены ::
main.cpp ::
$ <.exe/.elf> <file to process>
Для пункта 3 в качестве файла с полным спектром символов был взят архив .7z,
    так как заранее известно, что эти архивы обладают полноспектральной равномерной кодировкой.
    (Энтропия почти равна 8)
Так же были использованны файлы из ЛР3 и ЛР4
